Introduction

Word processing is an essential tool for office automation. Microsoft Word allows users to create, edit, format, and print documents efficiently. Unlike a typewriter, it allows instant editing and formatting, making documents more readable and professional.

Word processing involves creating and structuring content, applying formatting, and managing elements like tables, images, and templates.

Text Formatting

Text formatting controls the appearance of your text. Common features include:

  • Font family, size, color
  • Bold, Italic, Underline
  • Text effects and highlights
  • Alignment and spacing

Proper formatting improves readability and professional appearance of documents.

Paragraphs and Lists

Each paragraph can be aligned (left, center, right, justified) and spaced for clarity.

Lists help organize information:

  • Bulleted lists: unordered points
  • Numbered lists: ordered steps

Styles

Word offers predefined styles (Heading 1, Title, Normal) to ensure consistent formatting. Applying a style changes font, size, color, and spacing all at once.

Custom styles can also be created to match your preferred document layout.

Working with Tables

Tables organize data in rows and columns. Methods to insert tables include:

  1. Drag the grid to desired size
  2. Insert Table dialog to specify rows and columns
  3. Draw Table for custom layouts

Tables can be formatted using Table Tools for styles, borders, shading, and merging cells.

Templates and Document Structure

A template is a pre-designed document structure with predefined styles, layouts, and formatting. Using templates saves time and ensures consistency.

Mail Merge

Mail Merge automates personalized documents. It combines a main document (e.g., letter) with a data source (Excel, CSV, Outlook) to generate multiple customized copies.

  1. Start Mail Merge from the Mailings tab
  2. Select Recipients (existing file or new list)
  3. Insert merge fields where personalized data is needed
  4. Preview and Finish to generate all documents

Mail Merge saves time in creating multiple personalized letters or labels.

Summary

Mastering word processing allows ICT students to produce professional, well-structured, and formatted documents. Key skills include text formatting, using tables, templates, styles, and performing mail merge efficiently.
