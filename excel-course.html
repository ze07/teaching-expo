ICT Fundamentals – Practical Excel Training for L1 English Students

1. Introduction to Excel

Excel is a spreadsheet software used to organize data, perform calculations, create charts, and analyze information. Workbooks contain worksheets made of rows, columns, and cells.

Interface Overview

  • Ribbon: Home, Insert, Page Layout, Formulas…
  • Formula Bar: shows the content of the selected cell
  • Worksheet Grid: rows × columns
  • Sheets Tabs: Sheet1, Sheet2…

2. Worksheets, Rows, Columns, and Cells

Worksheets

A worksheet contains more than 1 million rows and 16,000 columns.

Cells

A cell contains text, numbers, dates, or formulas. Identified by column + row (example: A1).

Cell Pointer

  • Arrow → select
  • White cross → highlight
  • Black cross → fill handle
  • Double arrow → resize columns/rows

3. Working with Workbooks

  • Create new workbook: CTRL + N
  • Rename sheet: double-click the sheet name
  • Add sheet: + icon
  • Copy sheet: hold CTRL + drag
  • Delete: right-click → Delete

4. Selecting and Entering Data

Selecting data

  • Cell: click
  • Row: click row number
  • Column: click column letter
  • Range: click + drag
  • Whole sheet: CTRL + A

Entering values

  • Text: type and press Enter
  • Numbers: used in calculations
  • Dates: formatted automatically

5. Formulas and Functions

Basic Operators

  • Addition: =A1 + B1
  • Subtraction: =A1 - B1
  • Multiplication: =A1 * B1
  • Division: =A1 / B1

AutoSum

Click the result cell → AutoSum (Σ) → Enter.

6. Managing Rows and Columns

  • Resize: drag borders
  • AutoFit: double-click border
  • Insert: right-click → Insert
  • Delete: right-click → Delete
  • Hide/Unhide: right-click options

7. Freeze Panes

  • Freeze Top Row
  • Freeze First Column
  • Freeze Panes (custom)

8. Page Layout and Printing

  • File → Print → Preview
  • Page Orientation
  • Margins, Scaling
  • Headers and Footers
